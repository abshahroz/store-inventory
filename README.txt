<-- Store Inventory App -->

Store Inventory app is a simple app used to store weight of items in an inventory. 

Also it stores the date when an item was updated last. Users can update as well as delete the item

It is useful for small businesses as well as houses to keep track of their inventory and when it was updated last.

Users can start by installing this app on their android devices and clicking icon on main screen to start adding items.

If there is any problem, users can get help by commenting on this project.
